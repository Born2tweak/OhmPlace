Lab 2: Wireless Router
Overview: In this lab project, you will learn how to configure switches and routers in wireless environment.
Specifically, you will learn how to apply Virtual Local Area Network (VLAN), Dynamic Host Configuration
Protocol (DHCP), Port forwarding, Secure Shell (SSH), and Network Address Translation (NAT) in practice.

Install VyOS on mini passive-cooled pc with 6 ethernet ports and wireless networking. Use USB purchased
for class (mentioned in lecture).
Connect to the console port of the VyOS device serially with the speed of 115200 (or 9600 depending on the
device model) using PuTTY or SecureCRT.
The VyOS wireless router should set to a static Wide Area Network (WAN) interface of 44.34.XX.254 on
one of the Ethernet interfaces, with a default gateway of 44.34.XX.1 and subnet mask of 255.255.255.0
Configure Wireless network on VyOS machine including:
o Minimal Wireless Interface configuration.
▪ Internet Protocol (IP) address 172.16.XX.1 /24.
▪ description – “CNIT 344 Group XX Wireless Network”.
▪ wireless channel.
▪ wireless mode.
▪ wireless type – access point.
▪ wireless Service Set Identifier (SSID) – “gXXc344”.
▪ wireless security mode and password.
▪ wireless country code.

NAT will need to be used for all networks inside the private network. NAT will permit the traffic outside on
the WAN port.

and based on the port they connect to and have Internet access.
